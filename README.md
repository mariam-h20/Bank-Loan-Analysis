# Bank-Loan-Analysis

This dashboard provides a comprehensive analysis of bank loan performance using key financial and customer-related metrics.

## Project Overview

The goal of this project was to analyze loan applications, funding trends, borrower characteristics, and loan performance indicators to derive actionable insights for financial decision-making.

## Key Metrics Analyzed

- Total Loan Applications: 39K
- Total Funded Amount: $436M
- Total Amount Received: $473M
- Average Interest Rate: 12%

## Insights Extracted

### Loan Performance
- Good loans significantly outweigh bad loans, contributing over $435M in funded amount.
- Loan status breakdown shows Fully Paid, Current, and Charged Off categories with corresponding financial impact.
- Monthly Trend Analysis
- Loan funding and received amounts show a gradual decline across the months.
- This could indicate seasonality or changing lending policies.

### Loan Purpose Analysis
- Debt consolidation dominates loan purposes.
- Credit card refinancing and home improvement follow closely.

### Employment Length Impact
- Borrowers with 10+ years of employment represent the largest segment.
- Shorter employment duration correlates with smaller borrowing volume.

### Home Ownership Distribution
- Mortgage holders represent the largest funded segment.
- Renters also form a significant portion of loan recipients.

## Tools Used
- Power BI
- Data Cleaning & Transformation
- DAX Measures
- Data Modeling

## Business Value
This dashboard helps stakeholders:

- Assess loan portfolio health
- Identify high-risk segments
- Understand borrower behavior
- Monitor funding and repayment performance
